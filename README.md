## 📌 About Project
This project focuses on analyzing Python as an open-source software. It includes understanding its philosophy, license, ecosystem, and how it works in a Linux environment. Along with the report, five shell scripts are created to demonstrate practical Linux skills.

---

## 💻 Scripts Overview

### 1. System Identity Report
Displays system information like kernel version, user, uptime, and Python version.

### 2. Python Package Inspector
Checks whether Python is installed and shows its version.

### 3. Disk and Permission Auditor
Analyzes important directories related to Python and system.

### 4. Log File Analyzer
Reads a log file and counts occurrences of a keyword like "error".

### 5. Open Source Manifesto Generator
Takes user input and creates a personalized open-source philosophy statement.
